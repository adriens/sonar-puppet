Single quoted strings do not get interpolated. Thus, embedding variables in single quoted strings is useless. Strings containing variables should be surrounded with double quotes instead.

Noncompliant Code Example

warning 'Variable ${my_var} is undefined'

Compliant Solution

warning "Variable ${my_var} is undefined"

Notes

If the string is to be interpolated by another tool (and not by Puppet), to avoid confusion and make it explicit, prefer: 
$to_be_interpolated_by_another_tool = "\${git_branch}"
to 
$to_be_interpolated_by_another_tool = '${git_branch}'
A comment in this case might also make sense to clarify the intent.
