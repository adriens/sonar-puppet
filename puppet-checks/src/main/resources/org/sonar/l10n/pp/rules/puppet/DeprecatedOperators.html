The array appends operator (+=) is no longer supported since version 4.

Noncompliant Code Example

$a += ['b', 'd']

What to Use Instead

You can for example use the stdlib union function that returns a union of two arrays, without duplicates. For example, union(['a','b','c'],['b','c','d']) returns ['a','b','c','d'].
