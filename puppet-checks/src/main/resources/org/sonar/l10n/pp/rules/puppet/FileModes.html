File modes should be represented as 4 digits rather than 3, to explicitly show that they are octal values. File modes can also be represented symbolically e.g. u=rw,g=r.

Noncompliant Code Example

file { '/var/log/syslog':
  ensure => present,
  mode   => 644,
}

Compliant Solution

file { '/var/log/syslog':
  ensure => file,
  mode   => 'o-rwx',
}
