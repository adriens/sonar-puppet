In parameterized class and define declarations, required parameters should be listed before optional parameters (i.e., parameters with defaults).

Noncompliant Code Example

class ntp (
  $options   = "iburst",
  $servers,              # Noncompliant
  $multicast = false
) {}

Compliant Solution

class ntp (
  $servers,
  $options   = "iburst",
  $multicast = false
) {}
