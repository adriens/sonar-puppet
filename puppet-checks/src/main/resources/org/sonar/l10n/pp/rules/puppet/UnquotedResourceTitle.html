All resource titles must be quoted. If you are using an array of titles you must quote each title in the array, but cannot quote the array itself.

Noncompliant Code Example

file { foo:    # Noncompliant
  ensure => directory,
}

file { [
    abc,       # Noncompliant
    $var,
  ]:
  ensure => directory,
}

Compliant Solution

file { 'foo':
  ensure => directory,
}

file { [
    'abc',
    $var,
  ]:
  ensure => directory,
}

file { $foo:
  ensure => directory,
}
