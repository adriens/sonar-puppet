For a better understanding, resource default statements should be placed at the beginning of the file or at least prior to any resource declaration.

Noncompliant Code Example

class abc {

  file { '/tmp/foo.txt':
    ensure => present,
  }

  File {
    owner => 'user1',
  }

}

Compliant Solution

class abc {

  File {
    owner => 'user1',
  }

  file { '/tmp/foo.txt':
    ensure => present,
  }
}
