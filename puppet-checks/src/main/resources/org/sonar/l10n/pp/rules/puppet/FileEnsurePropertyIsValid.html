Whether the file should exist, and if so what kind of file it should be. Possible values are present, absent (also called false), file, directory, and link.

However, we recommend using link and target explicitly, since this behavior can be harder to read.

Noncompliant Code Example

file { '/tmp/foo':
  ensure => '/tmp/bar',
}

Compliant Solution

file { '/tmp/foo':
  ensure => link,
  target => '/tmp/bar',
}
