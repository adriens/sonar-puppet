Checks that ensure attributes of file resources are valid. Possible values are:

  • present
  • absent
  • false
  • file
  • directory
  • link

Noncompliant Code Example

file { '/var/log/syslog':
  ensure => pres,                 # Noncompliant: Invalid 'pres' value
}

file { '/var/log/syslog':
  ensure => '/var/log/messages',  # Noncompliant: 'link' value and 'target' attribute should be used instead
}

Compliant Solution

file { '/var/log/syslog':
  ensure => present,
}

file { '/var/log/syslog':
  ensure => link,
  target => '/var/log/messages',
}
