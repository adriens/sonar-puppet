It is needlessly complex to invert the result of a boolean comparison. The opposite comparison should be made instead.

Noncompliant Code Example

if !(a == 2) { }             # Noncompliant
$b = !(i < 10)               # Noncompliant
$c = !($a == 1 and $b == 2)  # Noncompliant

Compliant Solution

if a != 2 { }
$b = (i >= 10)
$c = $a != 1 or $b != 2
