All strings should be enclosed in single quotes, unless they contain variables or the following characters: " (double quotes), ' (single quote), \n, \r, \t. Note that if the string only contains a variable, quotes should be removed.

Noncompliant Code Example

abc { 'foo'
  text1 => "abc",                # Noncompliant, no variables, no special characters => single quotes should be used
  text2 => "Dollar sign is $",   # Noncompliant, no variables, no special characters => single quotes should be used
  text3 => "$var",               # Noncompliant, variable only => quotes should be removed
  text4 => "${var}",             # Noncompliant, variable only => quotes should be removed
}

Compliant Solution

abc { 'foo'
  text1 => 'var',
  text2 => 'Dollar sign is $',
  text3 => $var,
  text4 => $var,
  text5 => "My variable is ${var}",
  text6 => "blabla\n blabla\n",
}
