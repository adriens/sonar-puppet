Classes and defines must not be defined within other classes or defines. Classes and defines should be declared as close to node scope as possible. If you have a class or define which requires another class or define, graceful failures must be in place if those required classes or defines are not declared elsewhere.

Noncompliant Code Example

class apache {
  class ssl { }
}

class apache {
  define config() { }
}
