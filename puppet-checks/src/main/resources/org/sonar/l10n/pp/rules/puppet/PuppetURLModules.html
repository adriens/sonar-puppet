When using puppet:/// URLs, you should ensure that the path starts with modules/ (as the most commonly used mount point in the Puppet file server).

Noncompliant Code Example

file { '/etc/apache/apache2.conf':
  source => 'puppet:///apache/etc/apache/apache2.conf',
}

Compliant Solution

file { '/etc/apache/apache2.conf':
  source => 'puppet:///modules/apache2/etc/apache/apache2.conf',
}
