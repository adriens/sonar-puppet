This rule applies whenever an if statement is followed by one or more elsif statements, the final elsif shall be followed by an else statement. The requirement for a final else statement is defensive programming. The else statement should either take appropriate action or contain a suitable comment as to why no action is taken.

Noncompliant Code Example

if $is_virtual {
  #do_something
}
elsif $operatingsystem == 'Windows' {
  #do_something_else
}

Compliant Solution

if $is_virtual {
  #do_something
}
elsif $operatingsystem == 'Windows' {
  #do_something_else
}
else {
  # Do nothing because...
}

if $is_virtual {
  #do_something
}
elsif $operatingsystem == 'Windows' {
  #do_something_else
}
else {
  fail 'This case is not supported...'
}
