Shared coding conventions allow teams to collaborate efficiently. This rule checks that all classes and defines match the following regular expression: ^(::)?([a-z][a-z0-9_]*::)*[a-z][a-z0-9_]*$

Noncompliant Code Example

class my-class {}    # Noncompliant
define myDefine {}   # Noncompliant

Compliant Solution

class my_class {}
define my_define {}
