This rule checks that all classes and defines match the following regular expression: ^([a-z][a-z0-9_]*::)*[a-z][a-z0-9_]*$In shorts, it prevents using uppercase letters.

Noncompliant Code Example

class my-class {}    # Noncompliant
define myDefine {}   # Noncompliant

Compliant Solution

class my_class {}
define my_define {}
