Although node names should be one of the following per the Puppet Labs language reference manual:

  • A quoted string
  • The bare word default
  • A regular expression
unquoted names do not make the catalog compilation fail.

This rule check that node names are always quoted to comply with the language reference manual.

Noncompliant Code Example

node server1 {
}

Compliant Solution

node 'server1' {
}

See

  • Puppet Labs language reference manual for node definition
